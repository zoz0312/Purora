# Purora Backend

## 개요

- Mobile에서 전달받은 내용을 처리, 저장, 결과값 반환 등 역할을 수행합니다.
- 포로라 서비스의 전반적인 API를 제공합니다.

## 주요기능

### 파티관리

- 파티 조회
- 파티 생성
- 파티 삭제
- 특정 시간마다 파티 삭제 (유효시간 지난 경우)
- 파티에 유저 등록
- 파티에 유저 삭제

### 명령어관리

- 명령어 등록
- 명령어 조회
- 명령어 삭제

### 기타명령어

- 로또
- 연금복권
- 주식검색
- 날씨
- 주간날씨

### 포로라 서비스

- 회원 관리
- 라이엇 토큰 크롤러
- 사설 전적 관리
- 소환사 관리

### 관리자 명령어 [!]

- 관리자 명령어 Flag
- 관리자 유저 관리
- 허용된 방 관리
