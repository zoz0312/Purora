## Purora Client

### 개요

- Mobile에서 카카오톡 알림 중 명령어에 응답합니다.

## Purora Backend

### 개요
- Mobile에서 전달받은 내용을 처리, 저장, 결과값 반환 등 역할을 수행합니다.

### docker
 - docker-compose.forgit.yml 파일을 수정하여 실서버에 사용

## 운영 환경
- 카카오톡 <=> Mobile <=> Purora BackEnd <=> (추후 가능성)
